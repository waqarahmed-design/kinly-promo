# Kinly Promo

Promo video projects for the Kinly / Health Tracker app.

## Layout

- `promo-kinly/` — current promo (storyboard, design, capture, renders)
- `promo-video/` — earlier iteration
- `promo-video-v2/` — second iteration

Each subfolder is a self-contained HTML + GSAP scene with its own
storyboard, snapshots, and rendered output.

## Workflow

These are not running web apps — they are scripted scenes rendered to
video. Open the relevant `index.html` in a browser to preview; rendering
happens via the `capture/` or external recording tooling.

## Origin

Split out of the main `Health Tracker` repo on 2026-04-27 to keep the
mobile app, the marketing site, and promo deliverables in separate
repositories.
